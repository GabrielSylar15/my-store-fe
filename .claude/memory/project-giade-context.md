GIADE is a single-shop Vietnamese ecommerce site selling **everyday household items** (not tech/keyboard accessories — those products in the DB are test data only).

**Why:** Shapes all design and UX decisions — the audience is everyday Vietnamese shoppers, not tech enthusiasts.

**How to apply:** Design references should be local household/lifestyle brands, not Western tech boutiques. UI conventions should match what Vietnamese ecommerce shoppers expect (Shopee, Tiki, Lazada patterns).
